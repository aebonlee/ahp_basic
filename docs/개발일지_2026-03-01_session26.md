# 개발일지 — 세션 26

**날짜**: 2026-03-01
**목표**: 메인페이지에 통계 분석 섹션 추가 + 설문 및 통계 공개 페이지 & 상단 메뉴 추가

---

## 배경

현재 HomePage에는 핵심 기능(4개) → 프로세스(4단계) → CTA만 존재. 통계 분석 기능(10종)이 StatisticalAnalysisPage에 구현되어 있지만 메인페이지에서는 전혀 소개하지 않고 있었음. FeaturesPage의 extraGrid 패턴을 참고하여 통계 분석 카드 섹션을 추가.

---

## 작업 내용

### 1. HomePage.jsx — STAT_FEATURES 상수 & 섹션 추가

- 10개 통계 분석 카드 데이터 상수 `STAT_FEATURES` 추가
  - 기술통계, 독립표본 T검정, 대응표본 T검정, 일원분산분석, 카이제곱 검정
  - 상관분석, 단순선형회귀, 크론바흐 알파, 교차분석, Spearman 순위상관
- Process 섹션과 CTA 섹션 사이에 `<section id="statistics">` 삽입
- 5열 × 2행 카드 그리드, "자세히 보기 →" 링크로 /features 연결

### 2. HomePage.module.css — 통계 섹션 스타일

- `.statsSection` — 배경 #f8fafc, 보더, 패딩 100px 0
- `.statCardGrid` — CSS Grid `repeat(5, 1fr)`, gap 16px
- `.statCard` — 흰 배경, 보더, 둥근 모서리(12px), 호버 시 translateY(-4px) + 그림자
- `.statCardIcon` — 이모지 크기 1.6rem
- `.statCardTitle` — 0.9rem, 700, #0f172a
- `.statCardDesc` — 0.8rem, #64748b

### 3. 반응형 브레이크포인트

| 브레이크포인트 | 그리드 열 | 결과 |
|---------------|----------|------|
| > 1024px | 5열 | 2행 |
| ≤ 1024px | 3열 | 4행 |
| ≤ 768px | 2열 | 5행 |
| ≤ 480px | 1열 | 10행 |

### 4. SurveyStatsPage — 설문 및 통계 공개 소개 페이지 신규 생성

- **Hero**: "설문 및 통계" 제목, "Survey & Statistics" 태그
- **설문 설계 섹션 (SURVEY)**: 3개 상세 기능 카드
  - 4단계 설문 설계 (연구소개→동의→인구통계→커스텀)
  - 7가지 문항 유형 (단답형~리커트)
  - 응답 수집 & 집계 (실시간 모니터링, 자동 집계)
- **문항 유형 칩 바**: 7개 유형을 pill 형태로 시각화
- **통계 분석 섹션 (STATISTICS)**: 10종 카드 그리드 (5열×2행)
- **하이라이트 바**: 10종 / SPSS 대체 / Excel 내보내기 / 해석 가이드 — 4색 컬러
- **CTA**: 무료 시작하기 버튼
- 반응형: 5열→3열→2열→1열

### 5. 상단 네비게이션 메뉴 추가

- `PublicNav.jsx`의 NAV_LINKS에 `{ to: '/survey-stats', label: '설문 및 통계' }` 추가
- "주요 기능"과 "이용 가이드" 사이에 배치
- 데스크탑 & 모바일 메뉴 모두 자동 반영

### 6. 라우트 등록

- `App.jsx`에 `<Route path="/survey-stats" element={<SurveyStatsPage />} />` 추가
- lazy import 등록
