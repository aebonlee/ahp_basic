# 개발일지 — 세션 26

**날짜**: 2026-03-01
**목표**: 메인페이지에 통계 분석 섹션 추가

---

## 배경

현재 HomePage에는 핵심 기능(4개) → 프로세스(4단계) → CTA만 존재. 통계 분석 기능(10종)이 StatisticalAnalysisPage에 구현되어 있지만 메인페이지에서는 전혀 소개하지 않고 있었음. FeaturesPage의 extraGrid 패턴을 참고하여 통계 분석 카드 섹션을 추가.

---

## 작업 내용

### 1. HomePage.jsx — STAT_FEATURES 상수 & 섹션 추가

- 10개 통계 분석 카드 데이터 상수 `STAT_FEATURES` 추가
  - 기술통계, 독립표본 T검정, 대응표본 T검정, 일원분산분석, 카이제곱 검정
  - 상관분석, 단순선형회귀, 크론바흐 알파, 교차분석, Spearman 순위상관
- Process 섹션과 CTA 섹션 사이에 `<section id="statistics">` 삽입
- 5열 × 2행 카드 그리드, "자세히 보기 →" 링크로 /features 연결

### 2. HomePage.module.css — 통계 섹션 스타일

- `.statsSection` — 배경 #f8fafc, 보더, 패딩 100px 0
- `.statCardGrid` — CSS Grid `repeat(5, 1fr)`, gap 16px
- `.statCard` — 흰 배경, 보더, 둥근 모서리(12px), 호버 시 translateY(-4px) + 그림자
- `.statCardIcon` — 이모지 크기 1.6rem
- `.statCardTitle` — 0.9rem, 700, #0f172a
- `.statCardDesc` — 0.8rem, #64748b

### 3. 반응형 브레이크포인트

| 브레이크포인트 | 그리드 열 | 결과 |
|---------------|----------|------|
| > 1024px | 5열 | 2행 |
| ≤ 1024px | 3열 | 4행 |
| ≤ 768px | 2열 | 5행 |
| ≤ 480px | 1열 | 10행 |
