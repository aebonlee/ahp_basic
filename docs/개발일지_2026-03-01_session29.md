## 추가 수정 (세션 29-2)

### PublicNav.module.css — 네비 밑줄/로고 최종 수정

**네비 링크 밑줄 정리:**
- `text-decoration: none` 유지 (브라우저 기본 밑줄 제거)
- `::after` 밑줄 애니메이션 **복원** (hover/active 시 3px 하단 밑줄)
- `:hover`, `:focus`, `:active` 상태에도 `text-decoration: none` 명시

**로고 디자인 강화:**
- 배지 크기: `0.75rem` → `1rem`, 패딩 `3px 8px` → `6px 12px`
- 배지 배경: 단색 `#0046C8` → 그라데이션 `#0046C8 → #002E8A`
- 배지 라운드: `4px` → `6px`, box-shadow 추가
- 텍스트 크기: `1.15rem` → `1.35rem`, weight `700` → `800`
- 간격: gap `6px` → `10px`
- 앵커 밑줄: `:hover`, `:focus`, `:active` 상태 모두 `text-decoration: none`
