# 개발일지: Learn 페이지 레이아웃 및 공유 결과 페이지 반응형 개선

**작성일**: 2026-03-12
**작업 유형**: 스타일 개선
**상태**: 완료

---

## 1. 배경

Learn 페이지(`/learn`)의 2차 메뉴(탭 바, 사이드바)가 화면 전체 너비를 사용하여 일반 페이지 범위(1280~1600px)를 벗어나는 문제가 있었다. 또한 공유 결과 페이지(`SharedResultPage`)에 데스크톱/태블릿/모바일 반응형 레이아웃이 적용되지 않아 개선이 필요했다.

## 2. 변경 파일 목록

| 파일 | 변경 유형 | 설명 |
|------|-----------|------|
| `src/pages/LearnPage.module.css` | 수정 | 사이드바 항목 간격 + 탭 바 max-width 제한 |
| `src/pages/SharedResultPage.module.css` | 수정 | 3단계 반응형 레이아웃 적용 |

## 3. 구현 상세

### 3.1 Learn 페이지 사이드바 간격 조정 (`de90e32`)

- `.sidebar`에 `display: flex; flex-direction: column; gap: 4px;` 추가
- 2차 메뉴 항목 간 여백을 추가하여 가독성 향상

### 3.2 Learn 페이지 탭 바 너비 제한 (`186d74c`)

- `.tabBar`에 `max-width: 1400px; margin: 0 auto;` 추가
- 기존: 탭 바가 화면 전체 너비를 차지
- 변경: `.body` 영역(max-width: 1400px)과 동일한 너비로 정렬

### 3.3 공유 결과 페이지 반응형 (`ab5692a`)

| 브레이크포인트 | max-width | padding | 비고 |
|----------------|-----------|---------|------|
| 데스크톱 (>1024px) | 1024px | 32px 40px | 기본 레이아웃 |
| 태블릿 (≤1024px) | 1024px | 24px 24px | 좌우 여백 축소 |
| 모바일 (≤768px) | 1024px | 16px 14px | 폰트 축소, 여백 최소화 |

모바일에서 추가로 적용된 사항:
- `.projectName` 폰트: 1.35rem → 1.15rem
- `.banner` 폰트: 0.88rem → 0.82rem, 패딩 축소
- `.errorTitle` 폰트: 1.2rem → 1.05rem
