# 브레인스토밍 기능 사용 가이드 (원본 문서)

> I Make It 서비스의 브레인스토밍 기능 설명서 원문

## 개요

브레인스토밍 기능은 AHP 분석 모델을 수립하는데 도움을 주는 기능입니다. AHP 분석 모델이 이미 정해진 경우에는 브레인스토밍 기능을 활용하지 않고 바로 모델 수립 기능을 활용하시면 됩니다. 그러나 모델이 100% 신뢰할 수 있는 상태가 아닌 경우에는 브레인스토밍 기능을 이용하시면 도움이 됩니다.

## 사용 절차

### 1단계: 대안 입력
'대안' 입력 창에서 고려하고 있는 대안이나 가능성이 있는 대안의 후보를 입력합니다. 입력은 창의 아무 부분이나 클릭하여 입력하면 됩니다. 입력이 완료된 후, '입력' 이라는 단어를 클릭하거나 enter 키를 치면 입력창이 사라집니다.
- 수정이 필요한 경우에는 수정하려는 대상을 더블클릭 합니다.

### 2단계: 장점 입력
대안 중 하나를 선택하고(선택하지 않아도 되지만 특정하는 것이 장단점을 찾는데 도움이 됨), '장점' 입력 창에 해당 대안의 장점들을 쉬운 표현을 활용하여 입력합니다. 그런 다음 '단점' 입력 창에 단점들을 입력합니다.

### 3단계: 다른 대안의 장단점
다른 대안들을 차례대로 선택하고, '장점'과 '단점'을 입력합니다.

### 4단계: 판단 기준 도출
장단점이 모두 정리되었으면, 중요한 장단점을 맨 아래에 있는 '판단 기준' 패널로 끌어 놓습니다. 그러면 상자의 내용을 수정할 수 있는 상태가 되는데, 판단기준으로써 적당한 표현으로 수정합니다. 이런 방법으로 중요한 장단점을 모두 판단 기준으로 만듭니다.

- 필요한 경우 '판단 기준' 창의 빈공간을 클릭하여 새로운 판단기준들을 만들 수도 있습니다.

> **참고**: 앞서 선택한 대안을 통해 도출된 것과 동일한 장점과 단점은 별도로 입력하지 않아도 됩니다. 또한 어떤 대안에는 장점이 다른 대안의 단점이 되는 수가 있는데, 입력해도 되고 하지 않아도 됩니다. 장점과 단점을 정리하는 이유는 이를 활용하여 '판단 기준'을 찾기 위한 것이므로 굳이 중복되거나 반대되는 장단점을 모두 입력하지 않아도 됩니다.

### 장단점 → 판단기준 변환 예시
- '가격이 저렴하다'(장점) 또는 '가격이 비싸다'(단점) → **'가격'**으로 표현 수정
- '품질이 우수하다'(장점) 또는 '불량률이 높다'(단점) → **'품질'**로 표현 수정

### 5단계: 계층구조 구성
'가격'이라는 판단기준과 '유지비'라는 판단기준이 만들어져 있는 경우, '가격'도 초기에 지출되는 비용이므로, '비용'이라는 판단기준을 새로 만들고 '가격(구매비용)'과 '유지비'를 '비용'이라는 부모 판단기준 안으로 끌어 자식 판단기준이 되게 할 수 있습니다.

자식이 될 수 있는 판단 기준들을 부모가 될 수 있는 판단 기준 안으로 끌어 놓습니다. 손자 판단 기준은 자식 판단 기준 안으로 끌어 놓습니다.

### 6단계: 모델 검토 및 확정
이런 과정을 거쳐 1차 판단기준들, 그 속의 2차 판단기준들, 또 그 속의 3차 판단기준들의 배치가 모두 끝나면:
1. **'모델 보기'** 단추를 눌러 모델을 검토 및 수정합니다.
2. 검토가 완료되면 **'모델 확정'** 단추를 눌러 모델을 완성합니다.

## 계층구조 예시
```
목적 (프로젝트명)
├── 비용 (1차 기준)
│   ├── 가격(구매비용) (2차 기준)
│   └── 유지비 (2차 기준)
├── 품질 (1차 기준)
└── 디자인 (1차 기준)
```
