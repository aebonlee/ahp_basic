# 평가시작(Evaluation Start) 페이지 소스 분석

## 페이지 정보
- **URL**: default.aspx?key=main (관리자 메인으로 리다이렉트)
- **프로젝트 상태**: 평가중 (data-state='1')
- **역할**: 평가 시작 확인 후 메인 대시보드 복귀

## 핵심 발견사항

### 1. 서버 사이드 alert
- "평가를 시작하였습니다. 평가자들에게 평가를 시작할 수 있는 링크를 e-mail로 보내세요."
- 모델확정 → 평가시작 전환 시 표시

### 2. 프로젝트 상태 변경
- 이전: 대기중(6) → **평가중(1)**

### 3. 메인 페이지 콘텐츠 (#pm_main_div)
- 환영 메시지: "'더 좋은 판단과 선택'을 지원하는 I MAKE IT에 오신 것을 환영합니다."
- CEO 인사말: AHP 방법론 소프트웨어 소개
- 프로젝트 관리 과정 (4단계):
  1. 프로젝트 생성
  2. 모델 구축 및 평가자 추가/배정
  3. 평가 시작
  4. 결과 보기

### 4. JavaScript 로직
- sessionStorage 초기화: table_width=null, position_ahp=null
- 만료 체크: end값이 "0"이 아니면 자동 로그아웃
- mode_div를 main_holder 상단으로 이동 (prepend)

## 프로젝트 상태 코드 정리
| data-state | 상태 표시 | 설명 |
|-----------|----------|------|
| 1 | 평가중 | 평가 진행 중 |
| 2 | 생성중 | 모델 구성 중 |
| 6 | 대기중 | 모델 완료, 평가 시작 전 |
