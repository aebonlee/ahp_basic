AHP 방법론을 기반으로 한 의사결정 소프트웨어인 IMakeIt은 (주)디시전사이언스의 약 20년에 걸친 경험 및 know-how와 고객님의 의견을 기반으로 만들어진 소중한 결과물입니다.

"복잡한 것을 간단명료하게 만드는 것이 가치 있는 것이다!"

대부분의 경우, 겉모양이 화려하고 쓰지도 않을 수많은 기능을 가진 것을 선호하고 선뜻 대가를 지급합니다. 그러나 화려한 겉모양을 벗겨 내어 실체를 이해하고 수많은 기능을 익히기 위해서는 많은 노력이 필요합니다.

AHP 방법론 역시 난해한 설명을 이해하고 복잡한 과정을 거쳐야 하기 때문에 활용이 쉽지 않았습니다. IMakeIt은 논리 전개, 화면 배치, 사용자 인터페이스, 컬러 코딩, 기능 설명, 글자의 크기에 이르기까지 한국적 정서에 맞추어 편하게 사용하실 수 있도록 많은 논의와 피드백을 거쳐 다듬었습니다.

간결한 절차를 따라 하기만 하면 누구나 쉽게 AHP 분석 결과에 도달할 수 있습니다.

시작하기

'시작하기'를 활용하여 새로운 프로젝트를 추가하거나, 기존 프로젝트를 선택하여 관리/수정/삭제를 할 수 있습니다.

프로젝트의 상태

  1. 생성중 : 의사결정 모델 구성요소(기준, 대안) 및 평가자를 구성 중인 상태입니다.
  2. 대기중 : 의사결정 모델을 완료한 상태로 평가자 배정을 할 수 있습니다.
  3. 평가중 : 모델이 완료된 후 평가가 진행 중인 상태입니다.
  4. 평가종료 : 평가자들의 평가를 종료시킨 상태입니다.

평가 방법 설정

1차 기준들에 대한 평가 방법을 설정합니다. 기본 설정은 쌍대비교-실용입니다.

  1. 쌍대비교-실용 : 필요한 최소한의 쌍대비교를 실시하여 상대적 중요도를 도출합니다.
  2. 직접입력 : 데이터를 입력함으로써 중요도를 도출합니다.
  3. 쌍대비교-이론 : 이론상 필요한 모든 쌍대비교를 실시하여 상대적 중요도를 도출합니다.
