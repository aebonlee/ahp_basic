# copy_code

사용자가 웹에서 복제한 소스코드를 보관하는 폴더입니다.

## 폴더 구조 규칙
- 소스 제공 시 날짜_설명 형식으로 하위 폴더를 생성하여 저장
- 예: `2026-02-22_ahp_main_page/`
- 각 소스에 대한 분석 메모를 함께 보관
