# 인구통계학적 설문 편집 기능 개선 (2026-03-08)

## 변경 내용
- Step 3(인구통계학적 설문)이 기존 compact `QuestionCard` 대신 `GFormCard`(Google Forms 스타일)를 사용하도록 변경
- 기본 템플릿을 로드한 후에도 모든 질문을 자유롭게 편집 가능

## 이전 동작 (QuestionCard)
- 질문 카드를 **클릭해야** 유형 변경, 선택지 편집, 삭제/복제/이동 가능
- 클릭하지 않으면 읽기 전용 미리보기만 표시
- 연구자가 템플릿 로드 후 수정하기 어려움

## 변경 후 동작 (GFormCard)
- 질문 텍스트, 유형 셀렉터, 선택지 편집이 **항상 표시**
- 하단 액션바(복제/삭제/이동/필수 토글)가 항상 보임
- 설명 필드 추가 가능
- 우측 플로팅 툴바로 다양한 유형의 질문 빠르게 추가 가능
- Step 4(연구자 설문)와 동일한 UX 제공

## 수정 파일
| 파일 | 변경 |
|------|------|
