# AHP 학습 가이드 페이지 개발 (`/learn`)

**날짜:** 2026-03-09
**작업 유형:** 신규 기능 (Feature)

---

## 개요

연구자와 평가자가 AHP 방법론을 종합적으로 학습할 수 있는 `/learn` 학습 가이드 페이지를 신규 추가하였다.
기존 `/guide`(4단계 간단 소개)와 `/manual`(사용설명서)은 그대로 유지하며, 별도의 학습 전용 페이지로 구성하였다.

ahp_app 프로젝트의 6개 가이드 페이지(AHP 방법론, Fuzzy AHP, 연구자 가이드, 평가자 가이드, AI 활용, 종합 가이드)를 참조하여 콘텐츠를 한국어 데이터 배열로 변환하였다.

---

## 구조

### 5개 대분류 탭
| 탭 | 내용 | 서브 섹션 수 |
|---|---|---|
| AHP 방법론 | AHP 이론, 계층구조, 쌍대비교, 일관성검증, 가중치계산, 적용사례, 장단점 | 8개 |
| 연구자 가이드 | 연구방법론 개요, 이론적 배경, 연구설계, 계층구조 설계, 데이터수집, 분석방법, 타당성검증, 보고서작성 | 8개 |
| 평가자 가이드 | 평가자 개요, 평가시작, AHP이해, 쌍대비교방법, 일관성검토, 평가요령, 문제해결, 평가완료 | 8개 |
| AI 활용 가이드 | AI 개요, AI논문작성, 결과해석, 품질검증, 학술자료생성, 효과적사용법 | 6개 |
| Fuzzy AHP | 퍼지AHP 개요, 퍼지이론, 삼각퍼지수, 퍼지척도, 비퍼지화, AHP비교, 적용사례, 장단점 | 8개 |

### 레이아웃
- **상단:** Hero 배너 + 5개 대분류 탭 바 (가로 배치, 컬러 구분)
- **좌측:** 사이드바 (260px) — 서브 섹션 목록
- **우측:** 콘텐츠 패널 — 선택된 섹션의 상세 내용

### 반응형 (768px 이하)
- 사이드바 → 드롭다운 `<select>` 변환
- 카드 그리드 → 1열 변환
- 탭 아이콘 숨김

## 콘텐츠 블록 타입

| Type | 설명 |
|---|---|
| `text` | 제목 + 본문 텍스트 |
| `card-grid` | 2/3/4열 카드 그리드 (아이콘 + 제목 + 항목 리스트) |
| `table` | 테이블 (제목 + 헤더 + 행) |
| `list` | 번호/단계 리스트 (뱃지 + 제목 + 설명 + 수식) |
| `tip` | 파란색 팁 박스 |
| `warning` | 노란색 경고 박스 |

## 기술 결정

- **콘텐츠 데이터 분리:** JSX 비대화 방지를 위해 `learnData.js`로 데이터를 별도 분리
- **2단계 상태 관리:** `activeTab` (대분류) + `activeSection` (서브 섹션) 조합으로 콘텐츠 전환
- **CSS Modules 사용:** 프로젝트 기존 패턴과 일관성 유지
- **Lazy Loading:** App.jsx에서 `lazy(() => import(...))` 적용
