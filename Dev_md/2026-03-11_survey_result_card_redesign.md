# 설문 집계 상단 대시보드 카드 디자인 개선

**날짜:** 2026-03-11
**작업 범위:** `SurveyResultPage.jsx`, `SurveyResultPage.module.css`

---

## 개요

설문집계 페이지 상단의 대시보드 집계 카드(총 평가자, 설문 응답, 평가 완료, 진행중, 미시작 + 질문별 응답률) 디자인을 시각적으로 매력적으로 개선.

## 변경 내역

### 1. 메인 대시보드 카드 (dashCard) 개선

**기존:** 단순 흰색 박스 + 숫자 + 라벨
**개선:**
- 상단 3px 컬러 보더 (카드별 고유 색상)
- 아이콘 추가 (👥 총평가자, 📋 설문응답, ✅ 평가완료, ⏳ 진행중, ⏸️ 미시작)
- hover 시 shadow + translateY 효과
- 카드별 숫자 색상 차별화

**카드별 색상:**
| 순서 | 카드 | 보더/숫자 색상 |
|------|------|---------------|
| 1 | 총 평가자 | Indigo `#6366f1` |
| 2 | 설문 응답 | Blue `#2563eb` |
| 3 | 평가 완료 | Emerald `#059669` |
| 4 | 평가 진행중 | Amber `#f59e0b` |
| 5 | 미시작 | Red `#ef4444` |

### 2. 질문별 응답률 카드 (dashCardSm) 개선

**기존:** 단순 연한 배경 박스
**개선:**
- 좌측 3px Violet 보더 장식
- 퍼센트 표시 추가 (Violet 색상 mono 폰트)
- 바 색상 Violet으로 통일
- hover 시 shadow + translateY 효과

### 3. 프로그레스 바 개선

- 높이 5px → 6px
- 카드별 바 색상 자동 적용 (nth-child CSS)
