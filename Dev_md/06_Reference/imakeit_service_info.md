# 참고 자료 - I Make It 서비스 정보

## 서비스 개요
- **서비스명**: I Make It (AHP 분석 프로그램)
- **개발사**: (주)디시전사이언스 (Decision Science)
- **URL**: http://imakeit.kr
- **QuickStart**: http://imakeit.kr/info/pages/QuickStart/
- **연락처**: 02-707-1070 / serve@expertchoice.co.kr
- **웹사이트**: http://www.expertchoice.co.kr

## 서비스 특징
1. **웹 솔루션**: 설치 불필요, 브라우저 기반
2. **크로스 디바이스**: 핸드폰, 태블릿, 노트북, 데스크톱 모두 지원
3. **AHP 전문**: 정밀한 AHP(Analytic Hierarchy Process) 소프트웨어

## 접속 방식
- 관리자 링크: 고유 접속코드가 포함된 URL (예: `imakeit.kr/imk.aspx?makeit=XXXXX_username`)
- 평가자 링크: 평가자별 개별 링크 생성 → 이메일로 전송
- IE 11 이상 또는 Chrome 등 모던 브라우저 지원

## 핵심 기능 (복제 대상)
1. **프로젝트 관리**: 프로젝트 추가/관리
2. **기준(Criteria) 관리**: 기준 추가/편집
3. **대안(Alternative) 관리**: 대안 추가/편집
4. **쌍대비교 (Pairwise Comparison)**:
   - **쌍대비교 - 실용** (최소 판단, 기본값)
   - **쌍대비교 - 이론** (모든 쌍대비교)
5. **평가자 관리**: 평가자별 링크 생성 및 관리
6. **결과 분석**: AHP 가중치 계산, 일관성 비율(CR)
7. **Excel 다운로드**: 결과 내보내기

## 사용 흐름
1. 관리자가 프로젝트 생성
2. 기준(Criteria) 설정 (평가방법 선택: 실용/이론)
3. 대안(Alternative) 설정
4. 평가자에게 링크 전송 (또는 직접 평가)
5. 쌍대비교 평가 수행
6. 결과 확인 및 Excel 다운로드

## 개발 시 참고사항
- 스크립트 실행 차단 관련 처리 필요
- 반응형 디자인 필수 (모바일~데스크톱)
- 기간 만료 시 데이터 처리 정책 결정 필요 (Supabase 기반)
- 평가자 초대 기능 (이메일 링크 방식)
